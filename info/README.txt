					Scope of work done 

-----> Implemented requirement functionality: 

	- Build a graph on a formula provided by user.
		(formulas: "sin(x)", "cos(x)", "ln(x)", "x*x*x", "|x|", "sqrt(x)", "a*x*x+b*x+c") 
	- Build a graph on an array [x,y] provided by user. 
	- Have a grid, which may contain one or few graphs. 
	- Have an ability to scale a graph, redraw or delete it. 
	- Have an ability to change graphs properties (color, line type). 
	- Ability to save graphs as pictures to file. 
	- OS Linux Support.

-----> Not implemented requirement functionality: 

	- Rename graph.
